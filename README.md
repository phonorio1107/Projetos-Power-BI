# 📊 Projeto de Análise de Dados – Dashboard Financeiro

Este projeto foi desenvolvido com o objetivo de transformar dados financeiros brutos em informações estratégicas por meio de visualização de dados e análise exploratória.

O projeto simula um cenário corporativo em que uma empresa precisa organizar suas informações financeiras para entender melhor sua performance, identificar padrões e apoiar a tomada de decisão.

---

## 🎯 Objetivo do Projeto

Criar um dashboard que permita:

- Visualizar receitas, custos e lucro
- Identificar sazonalidade nas vendas
- Entender a participação dos custos na receita
- Facilitar a análise financeira da empresa

---

## 🛠️ Ferramentas Utilizadas

- Microsoft Excel
- Power BI
- Modelagem de dados
- Star Schema
- Tratamento e organização de dados

---

## 📂 Estrutura do Projeto

O projeto está organizado da seguinte forma:

/dados → base de dados utilizada  
/modelagem → estrutura de relacionamento entre tabelas  
/dashboard → arquivo final do dashboard  
/imagens → prints das visualizações

---

## 📈 Principais Análises

O dashboard apresenta:

- Receita total
- Custos totais
- Lucro
- Margem de lucro
- Comparação mensal de resultados
- Distribuição de custos

Essas análises permitem identificar padrões financeiros e compreender melhor o desempenho da empresa.

---

## 🧠 Aprendizados do Projeto

Durante o desenvolvimento deste projeto foram aplicados conceitos importantes de análise de dados:

- Organização e limpeza de dados
- Modelagem dimensional (Star Schema)
- Criação de métricas e indicadores
- Construção de dashboards interativos
